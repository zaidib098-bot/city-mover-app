# City Mover App

تطبيق يساعد المستخدمين على الانتقال للمدن الجديدة.  

## المميزات
- تسجيل دخول للمستخدم الباحث عن سكن والمؤجر.
- عرض المنازل المتاحة في دمشق (المناطق: كفرسوسة، مزة، أبو رمانة).
- عرض تفاصيل المنزل والخدمات القريبة على خريطة تفاعلية.
- إضافة منازل جديدة من قبل المؤجر.
